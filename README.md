# Tab-locker
Tab URL Locker is a Chrome extension that locks your active tab to its current URL, preventing any navigation, tab switching, new tabs, or new windows while the lock is active. It includes an optional countdown timer that keeps the lock enforced until time runs out — perfect for staying focused on a single page without distractions.

## Installation

1. **Download & unzip**  
   Download tab-locker.zip and extract it. You'll get a folder called tab-locker/.

2. **Open Chrome Extensions**  
   Type chrome://extensions/ in the address bar and press Enter.

3. **Enable Developer Mode**  
   Toggle Developer mode on in the top-right corner of the page.

4. **Load unpacked**  
   Click Load unpacked and select the extracted tab-locker/ folder.

5. **Pin to toolbar**  
   Click the 🧩 puzzle icon → find Tab URL Locker → click 📌 to pin it.
